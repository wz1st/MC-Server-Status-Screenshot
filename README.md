### 这是啥
**MC 服务器状态截图**     
利用mc server ping获取和客户端相似的服务器状态截图
#### 安装
```shell
git clone https://github.com/wz1st/MC-Server-Status-Screenshot.git
cd MC-Server-Status-Screenshot
pip install -r requirements.txt
```

#### 运行
```shell
python3 web.py
```

#### 访问
`https://xxx:5000/server/serip:25565`即可看到效果
